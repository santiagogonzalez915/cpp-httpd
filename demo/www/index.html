cpp-httpd

A C++17 HTTP/1.1 server built from scratch. Each link below exercises a different feature running on this process right now.

Demos

FeatureWhat it showsLink
Static files ETag, Cache-Control: max-age=3600, sendfile() → static demo

Request inspector CGI environment — proves the parser is working → inspector
Basic Auth .htaccess credential check, 401 + WWW-Authenticate → admin (admin / demo)
Directory listing Auto-generated listing via opendir/readdir → /files/

Implementation highlights

  • kqueue (macOS) / epoll (Linux) — O(1) per event, no fd ceiling
  • sendfile() — kernel moves file bytes to socket, zero user-space copies
  • 4-phase pipeline: FindConfig → Access → Content → Log
  • TLS 1.2+ via OpenSSL on a separate port (run make then scripts/gen_cert.sh)
  • Apache Combined Log Format → /tmp/cpp-httpd-demo-access.log
  • Catch2 unit tests (138 assertions), libFuzzer + ASAN on the HTTP parser
  • Benchmarked at 73% of nginx on 4 KB files; matches nginx on 1 MB
